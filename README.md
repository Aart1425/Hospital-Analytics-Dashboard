# 🏥 Hospital Analytics Dashboard using Power BI & SQL Server


---

# 📌 Project Overview

The **Hospital Analytics Dashboard** is an interactive Business Intelligence solution developed using **Power BI** with **SQL Server** as the primary data source.

The dashboard enables hospital administrators to monitor key performance indicators, analyze departmental performance, evaluate doctor contributions, and track financial metrics through dynamic visualizations.

---

# 🎯 Objectives

- Monitor hospital performance in real time
- Analyze admissions and patient trends
- Compare department revenue
- Evaluate doctor performance
- Monitor bill payment status
- Generate business insights for decision-making

---
